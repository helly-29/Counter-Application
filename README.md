# Counter Application (Flutter)

A simple **Counter App** built using **Flutter**.  
This application allows users to **increase** or **decrease** a counter value using buttons.

## Features

- Increment Counter
- Decrement Counter
- Clean and Simple UI
- Built with Flutter StatefulWidget

## Technologies Used

- Flutter
- Dart
- Material Design
